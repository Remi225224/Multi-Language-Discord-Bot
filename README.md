# Configuration

The configuration file named **.env** have to be created and replace **exemple.env** and content the token of your discord bot, his prefix and your id or the ids of all other owners. 
```  
# Bot configuration

PREFIX=BOT_PREFIX
TOKEN=BOT_TOKEN
OWNER=YOUR_DISCORD_ID
 ```

# Commands

**Installation**
> `npm install`

**Start the bot**  
> `node index.js`
